# CaCS_cement_hydration_dynamic

It consumes GEMS-exported aqueous activities (aáµ¢), converts them into ionic activity products, saturation ratios, and then integrates CSH nucleation/growth (channel A) and C3A systems with competitive adsorption &amp; internal partition (channel B).
